# KoiScript
A simple scripting language designed for extending C++ applications at runtime.

## Built-In Functions
### special
- assign
- print
### variable and function info
- size
### logical
- and
- or
- is_true
- not
- equals
- greater_than
- lesser_then
- greater_than_or_equal_to
- lesser_than_or_equal_to
### arithmetic
#### unary
- abs
- neg
- floor
- round
- ceil
#### binary
- add
- sub
- mul
- div
- mod
#### unary bitwise
- shift_left
- shift_right
- bit_or
- bit_xor
- bit_and
- bit_not
### control
- if
- while
